List the workflows, credentials, and recent runs that depend on a browser profile. Use this before deleting a profile to see what would be affected.

<CodeGroup>
```python Python
usage = await client.get_browser_profile_usage("bp_abc123")

for workflow in usage.workflows:
    print(f"{workflow.title} ({workflow.workflow_permanent_id}) via {workflow.via}")

for credential in usage.credentials:
    print(f"{credential.name} ({credential.credential_id})")

print(f"Recent seeded runs: {usage.recent_seeded_run_count}")
```

```typescript TypeScript
const usage = await skyvern.getBrowserProfileUsage("bp_abc123");

for (const workflow of usage.workflows) {
  console.log(`${workflow.title} (${workflow.workflow_permanent_id}) via ${workflow.via}`);
}

for (const credential of usage.credentials) {
  console.log(`${credential.name} (${credential.credential_id})`);
}

console.log(`Recent seeded runs: ${usage.recent_seeded_run_count}`);
```
</CodeGroup>

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `profile_id` | `str` | Yes | The browser profile ID. |
| `request_options` | `RequestOptions` | No | Per-request configuration (see below). |

### Returns `BrowserProfileUsage`

| Field | Type | Description |
|-------|------|-------------|
| `workflows` | `list[BrowserProfileUsageWorkflow]` | Workflows that reference the browser profile. |
| `credentials` | `list[BrowserProfileUsageCredential]` | Credentials attached to the browser profile. |
| `recent_seeded_run_count` | `int` | Number of recent runs that were seeded from this browser profile. |

**`BrowserProfileUsageWorkflow`**

| Field | Type | Description |
|-------|------|-------------|
| `workflow_permanent_id` | `str` | Permanent ID of the workflow. |
| `title` | `str` | Display title of the workflow. |
| `via` | `str` | How the workflow is linked to the profile (for example, `browser_profile_id` or `credential`). |

**`BrowserProfileUsageCredential`**

| Field | Type | Description |
|-------|------|-------------|
| `credential_id` | `str` | ID of the credential. |
| `name` | `str` | Display name of the credential. |
