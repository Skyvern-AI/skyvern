List all active browser sessions.

<CodeGroup>
```python Python
sessions = await client.get_browser_sessions()
for s in sessions:
    print(f"{s.browser_session_id} - {s.status}")
```

```typescript TypeScript
const sessions = await skyvern.getBrowserSessions();
for (const s of sessions) {
  console.log(`${s.browser_session_id} - ${s.status}`);
}
```
</CodeGroup>

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `request_options` | `RequestOptions` | No | Per-request configuration (see below). |
