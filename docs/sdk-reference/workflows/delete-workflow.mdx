Delete a workflow.

<CodeGroup>
```python Python
await client.delete_workflow("wf_abc123")
```

```typescript TypeScript
await skyvern.deleteWorkflow("wf_abc123");
```
</CodeGroup>

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `workflow_id` | `str` | Yes | The workflow version ID to delete. |
| `request_options` | `RequestOptions` | No | Per-request configuration (see below). |
