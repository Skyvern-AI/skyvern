Create a new page (tab) in the browser context.

<CodeGroup>
```python Python
page = await browser.new_page()
```

```typescript TypeScript
const page = await browser.newPage();
```
</CodeGroup>

Returns a `SkyvernBrowserPage` - a Playwright `Page` with AI methods added.
