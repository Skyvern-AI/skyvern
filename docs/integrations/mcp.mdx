The Skyvern MCP server lets you connect AI applications like Claude Desktop, Cursor, and Windsurf to browser automation.

Use it to fill out forms, download files, research information on the web, and moreâ€”directly from your AI assistant.

---

## What you'll need

- A local Skyvern deployment, or
- A Skyvern Cloud API key

---

## Option A: Setup Wizard

The interactive wizard configures your MCP client and sets up credentials automatically.

```bash
pip install skyvern
skyvern init
```

The wizard will:
1. Detect your MCP client (Claude Desktop, Cursor, or Windsurf)
2. Ask whether to use Skyvern Cloud or local mode
3. Configure your LLM provider and start the server (if local)

---

## Option B: Manual Configuration

For other MCP-enabled applications, add this to your MCP configuration file:

```json
{
  "mcpServers": {
    "Skyvern": {
      "env": {
        "SKYVERN_BASE_URL": "https://api.skyvern.com",
        "SKYVERN_API_KEY": "YOUR_SKYVERN_API_KEY"
      },
      "command": "PATH_TO_PYTHON",
      "args": [
        "-m",
        "skyvern",
        "run",
        "mcp"
      ]
    }
  }
}
```

For local mode, set `SKYVERN_BASE_URL` to `http://localhost:8000`. Find your local `SKYVERN_API_KEY` in the `.env` file after running `skyvern init`.
