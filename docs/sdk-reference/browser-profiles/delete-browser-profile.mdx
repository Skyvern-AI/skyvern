Soft-delete a browser profile. Any credentials attached to the profile are detached in the same transaction, so future logins for those credentials will not resolve to a deleted profile.

<CodeGroup>
```python Python
await client.delete_browser_profile("bp_abc123")
```

```typescript TypeScript
await skyvern.deleteBrowserProfile("bp_abc123");
```
</CodeGroup>

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `profile_id` | `str` | Yes | The browser profile ID to delete. |
| `request_options` | `RequestOptions` | No | Per-request configuration (see below). |

<Info>
To see which workflows, credentials, and recent runs depend on a profile before deleting, call [`get_browser_profile_usage`](/sdk-reference/browser-profiles/get-browser-profile-usage).
</Info>

<Warning>
`browser_profile_id` only works with `run_workflow`, not `run_task`. If you pass it to `run_task`, it will be silently ignored.
</Warning>

### Request options


Override timeout, retries, or headers for this call by passing `request_options` (Python) or a second options argument (TypeScript).

<CodeGroup>
```python Python
from skyvern.client.core import RequestOptions

request_options=RequestOptions(
    timeout_in_seconds=120,
    max_retries=3,
    additional_headers={"x-custom-header": "value"},
)
```

```typescript TypeScript
// Pass as second argument to any method
{
  timeoutInSeconds: 120,
  maxRetries: 3,
  headers: { "x-custom-header": "value" },
}
```
</CodeGroup>

| Option (Python) | Option (TypeScript) | Type | Description |
|-----------------|---------------------|------|-------------|
| `timeout_in_seconds` | `timeoutInSeconds` | `int` / `number` | HTTP timeout in seconds. |
| `max_retries` | `maxRetries` | `int` / `number` | Retry count. |
| `additional_headers` | `headers` | `dict` / `Record<string, string>` | Extra headers. |
| `additional_query_parameters` | - | `dict` | Extra query parameters. |
| `additional_body_parameters` | - | `dict` | Extra body parameters. |
| - | `abortSignal` | `AbortSignal` | Signal to cancel the request. |
| - | `apiKey` | `string` | Override API key. |
